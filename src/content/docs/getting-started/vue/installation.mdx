You can install Quirks via NPM

## NPM

```bash
$ npm i @nabla-studio/chain-registry @quirks/vue @quirks/store @quirks/wallets
# or
$ pnpm add @nabla-studio/chain-registry @quirks/vue @quirks/store @quirks/wallets
# or
$ yarn add @nabla-studio/chain-registry @quirks/vue @quirks/store @quirks/wallets
# or
$ bun add @nabla-studio/chain-registry @quirks/vue @quirks/store @quirks/wallets
```
