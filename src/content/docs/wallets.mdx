---
title: Wallets
---

Currently quirks supports several web wallets and also mobile (with wallet connect).

## Web Wallets

- [Keplr](https://www.keplr.app/)
- [Cosmostation](https://cosmostation.io/products/cosmostation_extension)
- [Leap](https://www.leapwallet.io/)
- [OKX](https://okx.com/)
- [Station](https://station.terra.money/)
- [xDefi](https://www.xdefi.io/)

## Mobile Wallets

- [Keplr](https://www.keplr.app/)
- [Leap](https://www.leapwallet.io/)
- [Cosmostation](https://cosmostation.io/products/cosmostation_mobile)
- Universal Wallet Connect, a generic provider for all wallets that support wallet connect, useful if there is not yet a specific implementation for a given wallet.
