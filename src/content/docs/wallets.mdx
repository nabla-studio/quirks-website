---
title: Wallets
---

Currently Quirks supports several web and mobile wallets (through 
Wallet Connect).

## Web Wallets

- [Keplr](https://www.keplr.app/)
- [Cosmostation](https://cosmostation.io/products/cosmostation_extension)
- [Leap](https://www.leapwallet.io/)
- [OKX](https://okx.com/)
- [Station](https://station.terra.money/)
- [xDefi](https://www.xdefi.io/)

## Mobile Wallets

- [Keplr](https://www.keplr.app/)
- [Leap](https://www.leapwallet.io/)
- [Cosmostation](https://cosmostation.io/products/cosmostation_mobile)
- Universal Wallet Connect, a generic provider for all wallets supporting 
Wallet Connect. It is particularly useful when there is no specific 
implementation for a specific wallet.
