title: Wallets
description: List of supported web and mobile wallets in Quirks.
---

Currently Quirks supports several web and mobile wallets (through
Wallet Connect).

## Cosmiframe

Repo: [https://github.com/DA0-DA0/cosmiframe](https://github.com/DA0-DA0/cosmiframe)

- [DAODAO](https://daodao.zone/)

## Web Wallets

- [Keplr](https://www.keplr.app/)
- [Cosmostation](https://cosmostation.io/products/cosmostation_extension)
- [Leap](https://www.leapwallet.io/)
- [OKX](https://okx.com/)
- [Station](https://station.terra.money/)
- [xDefi](https://www.xdefi.io/)
- [Shell](https://shellwallet.io/)
- [Leap Metamask Snap](https://www.leapwallet.io/snaps)

## Mobile Wallets

- [Keplr](https://www.keplr.app/)
- [Leap](https://www.leapwallet.io/)
- [Cosmostation](https://cosmostation.io/products/cosmostation_mobile)
- Universal WalletConnect, a generic provider for all wallets supporting
  WalletConnect. It is particularly useful when there is no specific
  implementation for a specific wallet.
